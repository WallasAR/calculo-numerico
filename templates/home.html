Cálculo Numérico

Implementações interativas de métodos numéricos fundamentais para resolução de problemas matemáticos computacionais.

Integração Numérica

Calcule integrais definidas usando os métodos clássicos do Trapézio e Simpson 1/3. Visualize pontos de avaliação, pesos e análise de erro para diferentes funções matemáticas.

Equações Não Lineares

Encontre raízes de funções usando Bisseção, Newton-Raphson e Ponto Fixo. Acompanhe a convergência passo a passo e analise a eficiência de cada método.

Interpolação de Newton

Construa polinômios interpoladores usando o método das diferenças divididas de Newton. Visualize a tabela de diferenças e avalie o polinômio em pontos específicos.

Tecnologias Utilizadas

Python 3 — Linguagem principal para implementação dos algoritmos.
Flask — Framework web minimalista para servir as aplicações.
SymPy — Biblioteca para computação simbólica e avaliação de funções.
Bootstrap 5 — Framework CSS para interfaces responsivas e modernas.
MathJax — Renderização de expressões matemáticas em LaTeX.
Jinja2 — Engine de templates para geração dinâmica de HTML.
